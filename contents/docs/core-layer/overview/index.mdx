This comprehensive documentation delves into the features and usage of the CleanBase.Core nuget . It is designed for crafting robust and maintainable applications that adhere to clean architecture principles. It provides essential services and tools for data management, external service integration, logging, and more.

**Nuget Structure:**

CleanBase.Core is structured into multiple layers, promoting separation of concerns:

- **Core:** The foundation of the framework, containing essential services like logging, identity management, and policy creation.
- **Data:** Handles all database interactions through repositories, transactions, and unit of work patterns.
- **External:** Provides integration with external systems and services such as message brokers and KeyVault.
